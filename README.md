# Flask Message Board

Proyek ini adalah aplikasi web sederhana menggunakan **Flask + SQLite**.  
Fitur:
- Home & About Page
- Tambah pesan baru (Add Post)
- Lihat semua pesan (All Posts)
- Database SQLite dengan inisialisasi via `flask init-db`
